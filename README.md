# Jaipur Bus route planning Simulator

## Overview
A C++ project that simulates city bus movement and route planning.

## Features
- Planning of the journey between two stations
- Dijkstra shortest path algorithm
## Technologies Used
- C++
- STL
- Object-Oriented Programming
- Graph Algorithms

## Future Enhancements
- GUI using SFML
- Real-time bus tracking
- Traffic simulation

## Prerequisites
- Before you begin, ensure you have the following prerequisites:

- C++ compiler 
- IDE (Integrated Development Environment) for compiling and running the code

## Installation
- Clone the repository to your local machine using:
- (https://github.com/akshu2307/Jaipur-bus-route-planning-Simulator.git)
- Open the project in your chosen IDE.
- Compile and run the project using the appropriate build and execution options in your IDE
